If you have a version of Microsoft Visual C Plus Plus Redistributable that does not allow you to install the MySQL server, download the updated version that can be found in the following link:

https://support.microsoft.com/en-us/help/3179560/update-for-visual-c-2013-and-visual-c-redistributable-package

Scroll down to the relevant file path of the 64-bit version of the program (e.g. English-United States). Then, install the file you have downloaded, and go back to the MySQL Installer window to install the MySQL Server.
